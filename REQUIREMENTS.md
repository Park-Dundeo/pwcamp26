## 백로그 (우선순위 순)

### 🔴 HIGH — mission.html UX 개선

#### REQ-01. 미션 순차 잠금 해제
- **요청:** 미션이 한꺼번에 다 보이면 재미없음 → 순서대로 하나씩 해금
- **흐름 (안):**
  ```
  [비밀번호 풀기 미션] → 해금 → [미션1: 경유지 인증샷] → 해금 → [미션2: 정탐 보고서]
  ```
- **세부사항 미결:**
  - 비밀번호 미션 형식 (숫자 코드? 단어? 힌트 방식?)
  - 비밀번호는 조별로 다른가, 공통인가
  - 해금 후 이전 미션으로 다시 올라가 볼 수 있는가

#### REQ-02. 모바일 반응형 전면 개선
- **요청:** 접속 기기(모바일/PC)에 따라 레이아웃 자동 변경
- **현황:** 일부 미디어쿼리 있으나 모바일 최적화 미흡
- **대상:** mission.html 전체 + admin.html 조별 그리드

### 🟡 MEDIUM — 콘텐츠 확정 필요

#### REQ-03. 로드뷰 힌트 이미지 삽입
- **요청:** 미션1 로드뷰 placeholder → 실제 네이버 로드뷰 스크린샷으로 교체
- **현황:** placeholder 텍스트만 있음
- **필요:** 1차 경유지 3곳(뚝섬유원지역/성수역/홍대입구역) 스크린샷 확정 후 삽입
- **조별 매핑:**
  - 1·2조 (전쟁기념관행): 뚝섬유원지역 근처
  - 3·4조 (서대문형무소행): 성수역 근처
  - 5·6조 (양화진행): 홍대입구역 근처

#### REQ-04. 촬영 미션 방식 확정
- **요청:** 1차 미션 촬영 방식 선택 필요
- **후보:**
  - 로드뷰 재현샷 (같은 앵글로 인증)
  - 특정 포즈 사진
  - 짧은 영상 (현재 구조상 용량 문제로 보류)
- **현황:** 로드뷰 재현샷 방향으로 진행 중이나 미확정
