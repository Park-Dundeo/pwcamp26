# 말씀 방향

본문 자체는 확정하지 않고, 방향과 느낌만 정리해 전도사님께 요청하기로 함. 후보로 언급된 본문: 로마서 12장, 에베소서 4장, 골로새서 3장, 히브리서 10장.

## 전도사님께 전달할 방향

- 예배 공동체의 재정비
- 함께 걸어온 시간을 돌아봄
- 하나님이 공동체 안에서 하신 일을 발견
- 서로를 축복
- 하반기를 다시 예배자로 시작

본문과 메시지 자체는 전도사님이 묵상하여 준비하는 것으로 역할 분담.

## 리트릿 내 예배·기도 흐름 (초안)

```
쉼 → Remember → Discover → 예배 → See Others → Dream Together → 기도 → Respond
```

- 중간 예배: "하나님이 우리 가운데 일하셨네"라는 발견에 대한 응답.
- 마지막 기도: 부담스러운 결심보다 서로를 축복하는 시간으로.
