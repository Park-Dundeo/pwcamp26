# 브랜딩과 굿즈

## 굿즈 방향

- 키워드와 굿즈를 연결해, 리트릿의 기억이 일회성으로 끝나지 않도록 함.
- 후보: 키링, 텀블러, 카드홀더, 스트랩, 팔찌.
- **최종 방향: 카드케이스** — 교통카드·체크카드·학생증 등으로 거의 매일 사용 가능.
- 굿즈는 예쁜 물건이 아니라 리트릿의 메시지를 반복해서 떠올리게 하는 장치로 정의.
- 매년 같은 디자인 시스템에 다른 키워드를 적용하면 찬양팀 리트릿의 문화가 될 수 있다는 방향 제시됨.

## 브랜딩 항목 (세부는 키워드 확정 후 진행)

- 메인 키워드: `ALIGN` / `RE:` (미확정)
- 메인 컬러
- 폰트
- 디자인 요소
- 카드케이스
- 포스터
- 초대장
- 숙소 공간 브랜딩

순서: 키워드 → 메시지 → 디자인 → 굿즈.
