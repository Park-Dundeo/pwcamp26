# 경험 설계와 프로그램

## 전체 구조

### 앞부분
- 이동
- 장보기
- 자유롭게 놀기
- 식사 준비

경험 목표: "오늘은 사역자가 아니라 함께 쉬는 공동체"

### 핵심 경험 (본문)
1. Remember: 우리가 함께 걸어온 길
2. Discover: 가장 기억나는 순간, 감사했던 순간, 하나님을 느꼈던 순간
3. See Others: 고마웠던 사람, 배운 사람, 힘이 되었던 사람 바라보기
4. Dream Together: 우리 팀에 더 많아졌으면 하는 모습, 새 친구가 느끼면 좋을 것
5. Respond: 나는 어떤 사람이 되고 싶은가

### 뒷부분
- 맛있는 것 먹기
- 놀기
- 산책
- 사진
- 게임
- 밤수다

전체 일정은 약 70%만 채우고 충분한 여백을 남기기로 함.

## 예배·기도 배치

```
쉼 → Remember → Discover → 예배 → See Others → Dream Together → 기도 → Respond
```

## 추가 기획 요소 (체크리스트)

Why / Experience / Keyword / Worship / Place / Rhythm / Scene / Symbol / Artifact / Follow-up / Host / Tempo

모든 요소를 판단하는 기준: "이 요소가 우리가 정한 경험을 실제로 만들어주는가?"

## 운영 시 리더 공통 체크리스트

- 혼자 있는 학생은 없는가?
- 너무 같은 사람끼리만 모여 있지 않은가?
- 하나님께 집중할 여백이 있는가?
- 일정이 너무 바쁘게만 흘러가고 있지는 않은가?
